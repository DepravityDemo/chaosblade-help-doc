# blade create container remove

# **Introduction**
remove a container
# **Flags**

```
--force
	force remove
--container-id
	Container id
--docker-endpoint
	Docker socket endpoint
--timeout
	set timeout for experiment

```

# **Example**

````
# Delete the container id that is a76D53933D3F
blade create docker container remove --container-id a76d53933d3f
````
